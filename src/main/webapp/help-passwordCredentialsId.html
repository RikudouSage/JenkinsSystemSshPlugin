If your key is not password protected, leave this empty.

Only the password will be used; if you specify username (or anything else) in the credentials it will be ignored.

You can either use the "Username with password" or "SSH Username with private key" credential types.

  • In "Username with password" specify the password in the "Password" box.
  • In "SSH Username with private key" specify the password in the "Passphrase" box.
